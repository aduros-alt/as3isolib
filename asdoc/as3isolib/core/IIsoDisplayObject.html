Interfacepublic interface IIsoDisplayObject extends INode, IEventDispatcherProxy, flash.events.IEventDispatcher, IContainer
ImplementorsIsoDisplayObject

The IIsoDisplayObject interface defines methods for any base display class needing rendering within an 3D isometric space.
