Packageas3isolib.bounds
Interfacepublic interface IBounds
ImplementorsPrimitiveBounds, SceneBounds

The IBounds interface defines the interface that all classes used bound-type information objects should use. Properties of IBounds implementors will refer to properties in isometric space.
