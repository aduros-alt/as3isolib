public static function drawIsoArrow(g:Graphics, originPt:Pt, degrees:Number, length:Number = 27, width:Number = 6, plane:String = "xy"):void

Draws an arrow in 3D isometric space relative to a specific plane.

Parameters
g:Graphics — The target graphics object performing the drawing tasks.
 
originPt:Pt — The origin pt where the specific drawing task originates.
 
degrees:Number — The angle of rotation in degrees perpendicular to the plane of orientation.
 
length:Number (default = 27) — The length of the arrow.
 
width:Number (default = 6) — The width of the arrow.
 
plane:String (default = "xy") — The plane of orientation to draw the arrow on.

Draws a rectangle in 3D isometric space relative to a specific plane.

Parameters
g:Graphics — The target graphics object performing the drawing tasks.
 
originPt:Pt — The origin pt where the specific drawing task originates.
 
width:Number — The width of the rectangle. This is relative to the first orientation axis in the given plane.
 
length:Number — The length of the rectangle. This is relative to the second orientation axis in the given plane.
 
plane:String (default = "xy") — The plane of orientation to draw the rectangle on.
