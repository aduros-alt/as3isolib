Interfacepublic interface IIsoScene extends IContainer, INode, IEventDispatcherProxy, flash.events.IEventDispatcher
ImplementorsIsoScene

The IIsoScene interface defines methods for scene-based classes that expect to group and control child objects in a similar fashion.
