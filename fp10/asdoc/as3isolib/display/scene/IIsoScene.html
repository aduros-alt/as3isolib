Packageas3isolib.display.scene
Interfacepublic interface IIsoScene extends IIsoContainer, INode, IInvalidation, IEventDispatcherProxy
Implementors IsoGroup, IsoScene

The IIsoScene interface defines methods for scene-based classes that expect to group and control child objects in a similar fashion.
