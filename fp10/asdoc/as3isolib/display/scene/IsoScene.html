layoutEnabledproperty 
public var layoutEnabled:Boolean = true

Flags the scene for possible layout rendering. If false, child objects are sorted by the order they were added rather than by their isometric depth.

layoutRendererproperty 
layoutRenderer:Object

The object used to layout a scene's children. This value can be either an IFactory or ISceneLayoutRenderer. If the value is an IFactory, then a renderer is created and discarded on each render pass. If the value is an ISceneLayoutRenderer, then a renderer is created and stored. This option infrequently rendered scenes to free up memeory by releasing the factory instance. If this IsoScene is expected be invalidated frequently, then persisting an instance in memory might provide better performance.
