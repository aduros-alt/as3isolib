BitmapFill()Constructor
public function BitmapFill(source:Object, orientation:Object = null, matrix:Matrix = null, colorTransform:ColorTransform = null, repeat:Boolean = true, smooth:Boolean = false)

Constructor

Parameters
source:Object — The target that serves as the context for the fill. Any assignment to a BitmapData, DisplayObject, Class, and String (as a fully qualified class path) are valid.
 
orientation:Object (default = null) — The expect orientation of the fill. Valid values relate to the IsoOrientation constants.
 
matrix:Matrix (default = null) — A user defined matrix for custom transformations.
 
colorTransform:ColorTransform (default = null) — Used to assign additional custom color transformations to the fill.
 
repeat:Boolean (default = true) — Flag indicating whether to repeat the fill. If this is false, then the fill will be stretched.
 
smooth:Boolean (default = false) — Flag indicating whether to smooth the fill.
